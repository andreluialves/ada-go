# Exercício 1 — primeiro teste

## Objetivo

Reconhecer e escrever as etapas Arrange, Act e Assert.

A função `isAdult` considera uma pessoa adulta a partir de 18 anos.

## Tarefas

1. Leia o teste existente e identifique as três etapas.
2. Crie um teste separado para `age = 17`, esperando `false`.
3. Crie um teste separado para `age = 18`, esperando `true`.
4. Execute `go test -v`.

Neste exercício, a repetição é intencional. Ela será tratada mais adiante.
